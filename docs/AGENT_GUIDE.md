## How to Use the Agents

### Via the Dashboard

1. **Navigate to the Agents Page**
   - Click "Agents" in the sidebar
   - You'll see a list of all available agents and past runs

2. **Start a New Agent Run**
   - Click the "Yeni Agent √áalƒ±≈ütƒ±r" (New Agent Run) button
   - Select the agent you want to use
   - Choose the project to analyze
   - Click "√áalƒ±≈ütƒ±r" (Run)

3. **View Results**
   - The agent will process your request (usually takes 10-30 seconds)
   - Once complete, click the "Eye" icon to view the results
   - You'll see all generated artifacts with full content
   - Artifacts are saved to your project for future reference

### Via the Project Detail Page

1. **Open Your Project**
   - Navigate to "Projelerim" (My Projects)
   - Click on any project card

2. **Run an Agent**
   - In the "Agents" section, click one of:
     - "Run Design Spec Agent"
     - "Run Workflow Agent"
     - "Run Content Agent"
   - The agent will run automatically
   - You'll be redirected to the results page

## Understanding Agent Artifacts

Each agent creates **artifacts** - structured deliverables that you can use in your project.

### Artifact Types

- **plan**: Strategic plans and specifications
- **task_list**: Action items and implementation steps
- **spec**: Technical specifications
- **copy**: Marketing and content copy
- **log**: Execution logs and metadata

### Artifact Metadata

Each artifact includes metadata:

- **Provider**: The AI provider used (e.g., "google")
- **Model**: The specific AI model (e.g., "gemini-2.5-flash")
- **Usage Stats**: Token counts, latency, costs (when available)

## Best Practices

### 1. Provide Complete Project Information

The better your project details, the better the agent outputs.

**Essential fields**:

- **Project Name**: Clear, descriptive name
- **Description**: Detailed explanation of what your project does
- **Sector**: Choose the most relevant sector
- **Project Type**: SaaS, Web App, Mobile App, etc.

### 2. Use Agents in Sequence

For best results, run agents in this order:

1. **Design Spec Agent** - Get the big picture
2. **Workflow Designer Agent** - Plan the technical details
3. **Content Strategist Agent** - Prepare your marketing

### 3. Review and Iterate

Agent outputs are starting points, not final deliverables:

- Review all generated content carefully
- Adapt suggestions to your specific context
- Use the "Wizard" to refine your project details
- Re-run agents after making significant changes

### 4. Save Important Artifacts

- Download or copy critical content to your project management tools
- Use artifacts as templates for your own documents
- Share agent outputs with your team for feedback

## Troubleshooting

### Agent Run Failed

**Possible causes**:

- Network connectivity issues
- AI service temporarily unavailable
- Invalid project data

**Solutions**:

- Check your internet connection
- Try running the agent again after a few minutes
- Verify your project has all required fields filled

### No Artifacts Generated

**Possible causes**:

- Agent run is still processing
- Agent encountered an error

**Solutions**:

- Wait and refresh the page
- Check the agent run status
- Contact support if the issue persists

### Artifacts are Generic or Low Quality

**Possible causes**:

- Insufficient project details
- Vague or unclear project description

**Solutions**:

- Add more detail to your project description
- Fill in all project fields (sector, type, etc.)
- Use the Wizard to provide structured input
- Re-run the agent with improved information

## Frequently Asked Questions

**Q: How long does an agent run take?**  
A: Most agent runs complete in 10-30 seconds, depending on the complexity of your project and the current AI service load.

**Q: Can I run multiple agents simultaneously?**  
A: Yes! Each agent run is independent. You can start multiple agents at once.

**Q: Are my agent outputs saved?**  
A: Yes, all artifacts are automatically saved to your project and can be accessed anytime from the Agents page.

**Q: Can I delete old agent runs?**  
A: Currently, all agent runs are preserved for reference. A deletion feature is planned for a future update.

**Q: Which AI model is used?**  
A: AYAZMA-ONE uses Google's Gemini 2.5 Flash, which provides fast, high-quality responses optimized for business planning tasks.

**Q: How much does each agent run cost?**  
A: AI costs are tracked in the Analytics section. Typical runs cost less than $0.01 per agent.
